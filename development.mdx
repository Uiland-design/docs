<Info>
  **Prerequisites**:
  - Node.js version 19 or higher
  - A docs repository with a `docs.json` file
</Info>

Follow these steps to install and run Mintlify locally.

<Steps>
<Step title="Install the Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="Preview locally">

Navigate to your docs directory where your `docs.json` file is located, and run the following command:

```bash
mint dev
```

A local preview of your documentation will be available at `http://localhost:3000`.

</Step>
</Steps>

## Custom ports

Mintlify runs on port 3000 by default. To use a different port, pass the `--port` flag. For example, to run on port 3333:

```bash
mint dev --port 3333
```

If the chosen port is already in use, Mintlify automatically uses the next available port:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Update the CLI

Each CLI release is tied to a specific version of Mintlify. If your local preview does not match the production site, update the CLI:

```bash
npm i -g mint@latest
```

## Validate links

Use the CLI to find broken links in your documentation:

```bash
mint broken-links
```

## Code formatting

Use editor extensions to recognize and format MDX. For VS Code, install the [MDX extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) for syntax highlighting and [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) for formatting.

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module using the darwin-arm64 runtime'>
    This error usually means your Node.js version is outdated. To resolve it:

    1. Remove the installed CLI: `npm remove -g mint`
    2. Upgrade to Node.js v19 or higher.
    3. Reinstall the CLI: `npm i -g mint`
  </Accordion>
  <Accordion title="Issue: Encountering an unknown error">
    Delete the `~/.mintlify` folder in your home directory, then run `mint dev` again.
  </Accordion>
</AccordionGroup>

To see what changed in the latest CLI version, check the [CLI changelog](https://www.npmjs.com/package/mintlify?activeTab=versions).
